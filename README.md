## Installation

- Clone this repository into your local machine:
```shell
git clone <your_repo>
```
```shell
cd <project folder>
```
```shell
composer install
```
- To run build in PHP web server:
```shell
composer serve
```
- Or run code on your PHP dev environment (Laragon, Xampp, Codespaces...)
